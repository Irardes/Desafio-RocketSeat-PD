# 🌍 Landing Page de Turismo

Projeto desenvolvido com **HTML** e **CSS** com o objetivo de criar uma **Landing Page de turismo** visualmente agradável, organizada e responsiva, destacando destinos turísticos, imagens atrativas e informações descritivas.

## 📌 Objetivo do projeto

Este projeto foi criado para praticar:

- Estruturação semântica com **HTML5**
- Estilização e organização visual com **CSS3**
- Criação de layout responsivo
- Separação entre estrutura (`HTML`) e estilo (`CSS`)

## ✨ Funcionalidades

- Menu de navegação fixo
- Seção principal com imagem de destaque
- Apresentação de destinos turísticos
- Cards com imagens e descrições
- Seção com dicas de viagem
- Área de contato
- Layout responsivo para diferentes tamanhos de tela

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
